Welcome to Weather Dashboard

Weather Dashboard is designed to provide users with accurate and up-to-date weather information. Our mission is to make weather data accessible and easy to understand for everyone.

Key Features

  • Real-time weather updates
  • Search and track multiple cities
  • Detailed weather information
  • Responsive design for all devices

How to Use

  1. Search for a city using the search bar
  2. View current weather conditions
  3. Add cities to your dashboard
  4. Track weather changes in real-time

Technology Stack

Built with modern web technologies:

  • Angular 19
  • TypeScript
  • SCSS for styling
  • Responsive design principles
